# Simoes-Costa Lab

Harvard Medical School • Boston Children's Hospital

The Simoes-Costa Lab studies how gene regulatory networks control cell fate decisions during vertebrate embryonic development.

Our research focuses on the regulatory mechanisms that drive cellular diversity, with particular emphasis on the **neural crest**, a multipotent stem cell population that generates many tissues of the vertebrate body.

To address these questions, we combine developmental biology with functional genomics and systems biology approaches, including epigenomics, single-cell sequencing, and regulatory network analysis.

This GitHub organization hosts the computational resources used in the lab, including analysis pipelines, project repositories, and shared infrastructure for reproducible genomics research.

## Main Repositories

- **core-pipeline** – shared pipelines and scripts used across projects  
- **template-project** – template for organizing new computational analyses  
- **legacy-repositories** – archived workflows retained for reproducibility
